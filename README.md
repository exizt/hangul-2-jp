# 한글 발음 -> 일본어
한글로 발음을 입력하면 일본어로 변환해주는 도구입니다. 

예: `니혼고` -> `にほんご`

<br><br><br>

# 링크

https://exizt.github.io/hangul-2-jp/

(위 링크에서 이용하실 수 있습니다)

# 동작에 대한 설명

한글로 발음을 적으면, 그에 맞춰서 변환을 해줍니다. 

변환된 결과에서 복사하셔도 되고, '복사' 버튼을 눌러서 변환된 결과를 복사하실 수 있습니다.
